# DOC MODE

Активируется, когда задача — написать документацию, README, схему, презентацию.

## Протокол
1. Узнай целевую аудиторию (разработчики? менеджеры? пользователи?)
2. Определи формат (Markdown? Mermaid? HTML-презентация?)
3. Составь структуру (оглавление) — покажи пользователю
4. Пиши по секциям, после каждой — уточняй, подходит ли стиль
5. В конце — проверь на согласованность терминов

## Критерии качества
- Есть примеры (код, скриншоты, диаграммы)
- Нет воды — каждое предложение несёт информацию
- Термины согласованы
- Есть "быстрый старт" для новых читателей
