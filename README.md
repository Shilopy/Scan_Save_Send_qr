# QR Сканер ЧЗ (Scan_Save_Send_qr)

Мобильное PWA-приложение для сканирования QR-кодов и DataMatrix Честного знака.

## Возможности

- **Сканирование** QR-кодов и DataMatrix через камеру телефона
- **Выбор категории**: Лёгкая промышленность / Строительные материалы
- **Автоматическое извлечение** первых 31 символа из каждого кода
- **Защита от дублей** — повторные коды не добавляются
- **Вибрация** + звуковой сигнал при успешном сканировании
- **Автосохранение** в localStorage браузера
- **Экспорт в .txt** с датой и временем
- **Копирование** всех кодов в буфер обмена
- **Отправка** через нативный Share API (Telegram, WhatsApp, email и т.д.)
- **История сессий** с возможностью восстановления
- **Тёмная/светлая тема**
- **PWA** — устанавливается на экран телефона как приложение
- **Работает офлайн** после первого открытия

## Установка на телефон

### Android (Chrome)
1. Откройте сайт в Chrome
2. Нажмите ⋮ (меню) → «Добавить на главный экран»
3. Или дождитесь баннера «Установить приложение»

### iOS (Safari)
1. Откройте сайт в Safari
2. Нажмите кнопку «Поделиться» (квадрат со стрелкой)
3. Пролистайте вниз → «На экран «Домой»»
4. Нажмите «Добавить»

## Использование

1. Выберите категорию товаров (Лёгкая промышленность / Строительные материалы)
2. Нажмите «Запустить камеру»
3. Наведите камеру на QR-код или DataMatrix
4. Телефон завибрирует — код добавлен в список
5. Сканируйте следующий код (камера перезапустится автоматически)
6. По окончании нажмите:
   - **💾 Сохранить .txt** — скачать файл с кодами
   - **📋 Копировать** — скопировать все коды в буфер
   - **📤 Поделиться** — отправить через любое приложение
   - **🗑 Очистить** — удалить все коды

## Технологии

- HTML5 / CSS3 / JavaScript (ES6+)
- [html5-qrcode](https://github.com/mebjas/html5-qrcode) — библиотека сканирования
- Web Audio API — звуковые сигналы
- Vibration API — вибрация
- Share API — нативная отправка
- Service Worker — офлайн-работа
- PWA Manifest — установка на экран
